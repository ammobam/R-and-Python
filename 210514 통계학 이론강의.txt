# 데이터 구성은 어떻게 정의하는가?
- 변수: 열의 이름
- 개체: 행의 이름
- 관측값: 값

# 데이터의 형태 구분
- 양적 데이터: 숫자로 표현된 데이터(수치형), 숫자(mean, ein, max등)/그래프로 요약
	-- 이산형 데이터
	-- 연속형 데이터
- 질적 데이터 : 속성 데이터(범주형), 표/그래프
	-- 순서형
	-- 명목형 : 순서X

# 데이터 요약
데이터의 왜곡 주의
- 심슨의 패러독스 (%의 함정)
	여러 하위 집단의 결과와 이들을 결합하여 한 집단으로 놓고 보았을 때의 결과가 다른 현상. 
	합계가 부분을 왜곡하거나 부분이 합계를 왜곡하는 경우를 의미.
	필요에 따라 데이터를 전체적으로 / 부분으로 나누어 분석해봐야 함.
	전체 데이터 수준에서 칼럼 비교할 때는 전체%를 비교해야함.
- 평균의 함정
	극단값으로 평균이 전체 데이터를 설명하기 어려워짐

# 데이터를 숫자로 요약 시
- 중심위치 : 평균, 중앙값, 최빈값
	-- 극단값의 영향을 받는 정도가 다름
- 산포도 : 분산, 표준편차, 
	사분위수(전체 데이터를 4등분으로 분할한 값), 
	변동계수(표준편차의 평균에 대한 상대비)
	사분위 범위(IQR, 상자수염그림), 
	범위
- 분포 : 왜도, 첨도

# 정규분포
- 평균을 기준으로 대칭을 이루는 데이터의 분포 형태
- 중심극한정리에 의해 데이터가 충분히 클 때 데이터의 분포는 정규분포를 따름
- 박스플롯과 겹쳐 이해하자
- 6시그마... 표준편차

## 표준화: 평균 0, 표준편차 1로 만들어주면 표준정규분포가 됨
- 비교 대상이 다른 분포, 또는 다른 단위를 가질 때 상대적인 크기로 비교하기 위해 사용
-- 미국 연봉, 한국 연봉 : 단위가 달러, 원으로 다르고 업종별 평균연봉이 다름
